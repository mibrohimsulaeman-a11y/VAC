# VAC JS wrapper

`vac-cli/` is the JavaScript package layer for launching the native VAC binary. It is not the Rust implementation and it is not a server runtime.

Current sandbox state:

- Source package name: `@vastar/vac-cli`.
- Native binary target after local build: `vac-rs/target/release/vac`.
- No public package or container publication is claimed by this checkpoint.
